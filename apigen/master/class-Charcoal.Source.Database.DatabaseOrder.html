# sql( )

Get the order's SQL string to append to an "ORDER BY" subquery.

Get the order's SQL string to append to an "ORDER BY" subquery.

There are 4 modes of "Order": - asc to order in ascending (A-Z / 0-9) order. - desc to order in descending (Z-A / 9-0) order. - rand to order in a random fashion. - values to order by a defined array of properties.
